Foxy Video Scraping: Real-Time Web Scraping, OCR, Data Labeling & Modeling with Export to Databases and Spreadsheets

Foxy is a versatile tool compatible with Linux, Windows, and macOS. It allows real-time web scraping, OCR (Optical Character Recognition), data labeling, and modeling, with export capabilities to databases and spreadsheets.

Tested on:

  • Linux: Ubuntu 24.04 LTS
  • Windows: Windows 11 Professional

Installation Guides

Windows

To install Docker on Windows, follow these steps:

  1. Download Docker Desktop:

  2. Visit the Docker Desktop for Windows page and download the installer.

  3. Install Docker Desktop:

  4. Run the installer and follow the instructions. This will enable Hyper-V and WSL 2 (Windows Subsystem for Linux) if they are not already enabled.

  5. Enable Hyper-V and WSL 2:

  6. Open Control Panel → Programs → Turn Windows features on or off.
  7. Check the boxes for Hyper-V and Virtual Machine Platform.

  8. Restart your computer.

  9. Install WSL 2:

  10. Open PowerShell as an administrator.
  11. Run wsl --install to install WSL and set version 2 as the default.

  12. Restart your computer if necessary.

  13. Configure Docker Desktop to Use WSL 2:

  14. Open Docker Desktop.
  15. Go to Settings → General and ensure "Use the WSL 2 based engine" is enabled.

  16. Under Resources → WSL Integration, enable integration with your Linux distributions.

  17. Verify the Installation:

  18. Open a terminal (cmd, PowerShell, or WSL).
  19. Run docker --version to confirm Docker is installed correctly.
  20. Run docker run hello-world to test Docker. You should see a welcome message.

Mac OS

To install Docker on macOS, follow these steps:

  1. Download Docker Desktop for Mac:

  2. Visit Docker Desktop for Mac and download the installer.

  3. Install Docker Desktop:

  4. Open the downloaded .dmg file and drag the Docker icon to the Applications folder.

  5. Open Docker Desktop:

  6. Go to the Applications folder and double-click Docker to start it.

  7. Verify Installation:

  8. Open a terminal and run docker --version.
  9. Run docker run hello-world to verify the installation.

Linux

To install Docker on Linux, refer to the Docker Linux Install Guide.
